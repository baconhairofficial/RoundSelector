# RoundSelector

A lightweight Bloons TD 6 Mod Helper mod.

## Features

- Press **J** during a match
- Enter a target round
- Instantly jump to that round

## Requirements

- Bloons TD 6
- MelonLoader
- BTD6 Mod Helper

## Installation

Place `RoundSelector.dll` inside your BTD6 `Mods` folder.

## Usage

Start a match, press **J**, enter a round number, and confirm.
